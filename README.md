## ⚙️ Requirements
1. **Neovim**:
   ```bash
   brew install neovim
   ```
2. **Alacritty**: Terminal Emulator 
   ```bash
   brew install alacritty
   ```
3. **Tmux**: Terminal Multiplexer 
   ```bash
   brew install tmux
   ```

---

```
.config
```
